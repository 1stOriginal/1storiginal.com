Actually the reason behind is as the dimensions are constant. Especially when you have group selfie and your friend’s face get cut down. So here are the lists of resolutions. Probably top 7 social media and their dimensions in profile picture and cover picture.

Twitter Cover and Profile Picture Size

  • Header Image – 1500 x 500
  • Profile Photo – 400 x 400
  • Post Image – 1024 x 512 (displays inline as 506 x 253).

Facebook Cover and Profile Picture Size

  • Cover Photo – 851 x 315
  • Profile Photo – 300 x 300 (displays as 160 x 160)
  • Post Link Image – 1200 x 630 (600 x 315 min)
  • App Icon – 111 x 74.

Google+ Cover and Profile Picture Size

  • Cover Photo – 2120 x 1192 (1080 x 608 min)
  • Profile Photo – 250 x 250
  • Post Image – 886 full, 497 single column (height is scaled for you).

Instagram Profile Picture Size

  • Recommended 1:1 ratio with a minimum of 600px* width, 1200px recommended. Landscape images with a ratio between 1.91:1 and 4:5 are also supported as of 2015.
  • *If creating an Instagram ad, minimum 1200px width is recommended so it can also be run on Facebook.

Linkedin Cover and Profile Picture Size

  • Profile Background – 1400 x 425
  • Profile Photo – 200 x 200
  • Company Banner – 646 x 220 min
  • Company Logo – 100 x 60
  • Company Hero – 974 x 330 min
  • Company Square Logo – 50 x 50.

YouTube Cover and Profile Picture Size

  • Cover Photo – 2560 x 1440
  • Video Thumbnail – 1280 x 720.

Pinterest Cover and Profile Picture Size

  • Post Image – 2:3 or 1:3.5 ratio & 600 min (displays as 216 board, 236 feed, 736 single)
  • Pinterest is depended upon image ratio. 2:3 (ex: 600 x 900) or 1:3.5 (ex: 900 x 2100)

Skype Profile Picture Size

After searching and doing a few experiments, I have found that the size is not 96×96, it is not 109×109. Although if you make your picture 109×109, the image seems to be too big. What I have found is that the exact size, is 480×480. You can look at my Skype picture if you want clarification.

Tumblr Cover and Profile Picture Size

Post Image – 1280 x 1920 (500 x 750 min before scaling).

Flickr Cover and Profile Picture Size

Profile Photo – 300×300

Flickr Cover Photo Dimensions – 2048×492.
